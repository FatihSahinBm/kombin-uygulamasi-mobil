📁 DOSYA YAPISI VE MİMARİ

Kodları düzenli ve modüler yaz:

JavaScript:
Tüm kodlar tek dosyada toplanmamalı
İşlevlere göre ayır:
main.js → ana kontrol merkezi
api.js → API işlemleri
ui.js → DOM işlemleri
utils.js → yardımcı fonksiyonlar
main.js, diğer dosyaları yöneten ana dosya olmalı
CSS:
Genel stil dosyası (style.css)
Gerekirse bileşen bazlı ayır (örneğin: components.css)
HTML:
Sayfalar anlamlı şekilde ayrılmalı (örnek: index.html, result.html)
🔗 API KULLANIMI
API çağrıları ayrı bir JS dosyasından yapılmalı (api.js)
API key gibi hassas bilgiler:
Ayrı bir dosyada tutulmalı (örnek: config.js)
Doğrudan kod içine gömülmemeli
🗄️ BACKEND (SUPABASE)
Backend olarak Supabase kullanılacak
Şunları yap:
Gerekli tabloları kendin tasarla ve isimlendir
Örnek: users, preferences, outfits
Tablolar arasında mantıklı ilişkiler kur
Supabase bağlantısı ayrı bir dosyada yönetilmeli
